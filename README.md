
# Crypto App

## App features:
- Live cryptocurrency data
- Saving current user's portfolio
- Searching, Filtering, Sorting, and Reloading data
- Custom color theme and loading animations

## Technical features:
- MVVM Architecture
- Core Data
- FileManager
- Combine
- Multiple API calls
- Codable
- 100% SwiftUI interface
- Multi-threading
